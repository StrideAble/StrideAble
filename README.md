# StrideAble Website

A modern, interactive, GitHub-ready static landing page for **StrideAble** — an AI-powered attachment that transforms walkers, canes, crutches, wheelchairs, and scooters into intelligent mobility companions.

Built for the **Governor's School Business Pitch Competition**.

---

## About StrideAble

StrideAble is a compact AI-powered device that clips onto existing mobility aids, turning them into intelligent safety systems. It features:

- 🎯 **Geometric AI Camera** — front-facing obstacle detection
- 📳 **Haptic Feedback** — vibration alerts through hand pads
- 🛡️ **Fall Detection AI** — automatic caregiver alerts
- ❤️ **Health Monitoring** — continuous wellness tracking
- 📱 **Caregiver App** — real-time safety and location updates
- 🤖 **AI Companion Chatbot** — personalized reminders and support
- 🔦 **Flashlight Safety** — automatic low-light visibility

## Opening Locally

1. Download or clone the project folder
2. Make sure your images are in the `images/` folder with the correct names
3. Double-click `index.html` to open it in your default browser
4. All features work without a server — no setup required

---

## Uploading to GitHub

1. Create a new GitHub account at [github.com](https://github.com) if you don't have one
2. Click **"New repository"**
3. Name it: `strideable-website`
4. Set it to **Public**
5. Click **"Create repository"**
6. Click **"uploading an existing file"** or use the GitHub Desktop app
7. Drag and drop all project files and folders into the upload area
8. Make sure to upload the entire folder structure including `css/`, `js/`, and `images/`
9. Click **"Commit changes"**

---

## Publishing with GitHub Pages

1. Go to your repository on GitHub
2. Click the **Settings** tab (gear icon)
3. Scroll down and click **Pages** in the left sidebar
4. Under **Build and deployment**, set:
   - **Source:** Deploy from a branch
   - **Branch:** `main`
   - **Folder:** `/ (root)`
5. Click **Save**
6. Wait 1–2 minutes for GitHub to build the site
7. Refresh the page — your live URL will appear at the top
8. Share your GitHub Pages link!

Your site will be live at:
`https://YOUR-USERNAME.github.io/strideable-website/`
